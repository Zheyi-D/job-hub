ℹ 关于 JobHub

        
      

      

        
JobHub 帮你做三件事：填简历、记投递、看数据。


        
📝 简历填充 — 把内容一键填进网页表单

        
投简历时打开侧边栏，你的简历信息（姓名、学校、经历等）都在这。在网页上点一下想填的输入框，再点侧边栏里对应的字段，内容就自动填进去了。

        
        
点 「🤖 AI」可以上传简历文件（Word/PDF/文本），AI 自动识别内容并填好。也支持粘贴简历文本或跳转到 DeepSeek、豆包、Kimi 用外部 AI。AI 功能需在设置页配置 API Key（支持 DeepSeek / Kimi / 豆包）。

        
点 「✏️ 编辑」可修改简历内容、增删字段、拖拽排序，改完点 「💾 保存」。左上角下拉框可创建多份简历（比如投产品方向和投技术方向各一份），点旁边 ⚙ 管理。


        
📋 投递追踪 — 记录投递并同步到飞书

        
打开岗位页面，切换到投递追踪 Tab，插件会自动抓取公司名、岗位、链接。核对无误后点 「保存到飞书」，就写进你的飞书多维表格了。

        
时间默认当前、状态默认「已投递」，都可以改。同一个链接不会重复记录。切换网页标签时会自动识别新页面。

        
需要在设置页配置飞书应用凭证才能同步，步骤见设置页的「首次配置指南」。


        
📊 看板 — 投递数据一目了然

        
自动统计：总投递数、进行中的、面试中的、拿到 Offer 的数量，还有各状态的分布图和近 7 天趋势。数据来自投递追踪记录，不需额外维护。


        
⚙ 设置页

        
设置页包含 AI API Key 配置、飞书凭证、投递历史管理、简历数据导入导出等。点击底部 「⚙ 设置」打开。
