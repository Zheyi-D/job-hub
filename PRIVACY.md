# JobHub 隐私政策

**生效日期：2026 年 7 月 20 日**

## 数据收集

JobHub 扩展**不收集、不存储、不传输**任何用户个人信息到外部服务器。

## 本地存储

以下数据仅保存在你本机的浏览器扩展存储中（Chrome/Edge 的 `chrome.storage`），不会发送到任何第三方：

- 简历填写内容（用户自行编辑的数据）
- 投递记录历史
- 飞书 API 凭证（App ID、App Secret）
- AI API Key 配置

## 第三方服务

当用户主动选择使用以下功能时，数据会发送到用户指定的服务：

- **飞书同步**：投递记录发送到用户自己配置的飞书多维表格（通过飞书 Open API）
- **AI 简历解析**：简历文本发送到用户自己配置的 AI 服务商（DeepSeek / Kimi / 豆包 等）

以上服务均由用户自行配置，JobHub 不参与数据传输，也不将数据发送到开发者的服务器。

## 权限使用

扩展请求以下浏览器权限，全部用于实现核心功能：

| 权限 | 用途 |
|------|------|
| `sidePanel` | 侧边栏操作界面 |
| `storage` | 本地数据持久化 |
| `activeTab` | 识别当前网页标签 |
| `scripting` | 向网页注入表单填充脚本 |
| `<all_urls>` | 表单填充和投递信息抓取 |
